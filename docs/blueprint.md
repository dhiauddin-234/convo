# **App Name**: Convo

## Core Features:

- User Authentication: Secure user registration and login using Firebase Authentication with email and password.
- User Discovery: Allow users to search for and find other users within the application.
- One-to-One Chat Rooms: Enable private, direct chat rooms between two users.
- Real-time Messaging: Send and receive text messages in real-time using Firestore.
- Message Persistence: Store all messages in Firestore for persistent chat history.
- Online/Last Seen Status: Display user online status or last seen timestamp.
- Message Moderation Tool: Moderate chat message content with the Genkit. The tool analyzes the user-provided chat text and flags if it violates platform guidelines.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to evoke trust and stability.
- Background color: Very light blue (#E8EAF6) to ensure readability and a calming atmosphere.
- Accent color: Vibrant orange (#FF9800) for calls to action and important notifications.
- Font pairing: 'Inter' (sans-serif) for body text and 'Space Grotesk' (sans-serif) for headlines.
- Use clean, simple vector icons for a modern look.
- Design a responsive layout that adapts to different screen sizes for optimal viewing on all devices.
- Incorporate subtle animations for loading messages and updating the online status.
